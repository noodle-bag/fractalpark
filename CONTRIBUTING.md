# Contributing

Thanks for your interest in FractalPark.

## Development

1. Fork the repository and create a feature branch.
2. Install dependencies with `npm install`.
3. Run `npm run lint` and `npm run test:run` before opening a pull request.
4. Keep user-facing text in `messages/en.json` and `messages/zh.json`.
5. Add focused tests when changing parsing, URL state, gallery data, rendering contracts, or saved fractal behavior.

## Pull Requests

Please keep pull requests focused. Include a short description of the behavior change, screenshots for UI changes when helpful, and any manual verification notes.
